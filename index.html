The main focus for this site is to have interactive possibilities with Graphs and Data Tables. Comment from the Developer

This site makes use of the official World of Tanks API. You can search by player name and can use various filter options to see that players statistics per Tier, nation and Tank Type. Statistics are as accurate as WoT API is up-to-date and is only useable for the EU server at the moment! This site uses also the formula for WN8. calculation

How to Use this site

This site differs from the existing sites because it has some interactive charts which can be used to show statistics based on a subset of data. For example you can select 1 or more Levels, Nation or tank types and see your stats only for those tanks. In order to do so you can use the "Select" boxes or click on the specific bar or pie-piece in the charts. All data (table and charts) are automatically adjusted! The reason why i am offering this functionality is because people can excel on lower levels and suck on the higher levels while the overall statistics don't show.
